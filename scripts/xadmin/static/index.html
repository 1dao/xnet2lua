关于 xadmin

xadmin 是基于 xnet2lua 的轻量管理控制台，通过 NATS 在多个 SERVER_NAME 进程间路由脚本执行。

  • 启动：bin/xnet scripts/xadmin/xadmin_main.lua SERVER_NAME=xadmin1
  • 账户登录：首次打开自动进入初始化页，配置 MySQL 连接与默认管理员；之后所有接口需登录会话。
  • 数据库配置保存在 xadmin_db.json（可用 XADMIN_DATA_FILE 覆盖路径），账户与会话存于 MySQL。
  • 也可在 xnet.cfg 用 XADMIN_DB_HOST/PORT/USER/PASSWORD/DATABASE 预置连接；有值即覆盖 JSON（按字段），五项齐全时初始化页只需设管理员账户。
  • 兼容旧版 XADMIN_TOKEN：设置后也可用 X-Xadmin-Token 头免登录调用接口。
